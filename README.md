# RouterHaus – Static Site

## Development
- `npm run build` copies `input.css` to `docs/main.css`.
- `npm run lint` runs a syntax check on JavaScript files.
- `npm test` validates JSON data files.

To preview locally:
```bash
python -m http.server -d docs
```
